# wikipedia-modernity
## What is it?
- This is a domain-specific, AI-enhanced tool for browsing wikipedia content.
- The idea is to have create embeddings from wikipedia for a certain domain, say "Modernity". That can be done by using PetScan tool to get a number of articles to embed.
- Using those embeddings, we can do interesting, useful things. For instance, semantic search, something like "Where is the topic of 'becoming-machine' most prominent?
- We could weave together unexpected trails of thought, like Deleuzian rhizomatic traversal.
